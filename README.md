# Python System Info Identifier

## Project Description
This project is a Python-based system information identifier that provides a comprehensive overview of the system's hardware and software specifications. It is designed to assist users in assessing their system's capabilities and performance metrics easily.

## Features
- Retrieves system information including CPU, RAM, Disk Space, and OS details.
- Provides user-friendly output formatting.
- Compatible with multiple operating systems including Windows, macOS, and Linux.
- Easy integration with existing Python applications.

## Installation Instructions
1. Ensure you have Python 3.6 or above installed on your system.
2. Clone this repository:
   ```bash
   git clone https://github.com/inknskinfreak935/System--info.git
   ```
3. Navigate to the project directory:
   ```bash
   cd System--info
   ```
4. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage Examples
To use the System Info Identifier, simply run the following command in your terminal:
```bash
python system_info.py
```
This will output the system information in a readable format.

## Requirements
- Python 3.6+
- Required libraries listed in the `requirements.txt`
